Button Spinners

Indicate the loading state of a button.

These “spinners” can be used to show the loading state in your projects. There is some JavaScript involved and the whole thing is quite unhinged. Their appearance, alignment, and sizing can be easily customized utility classes, if you write the corresponding utility styles yourself.

For accessibility purposes, you probably shouldn't use those, at all.
